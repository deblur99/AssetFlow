# AssetFlow — Cursor 컨텍스트

이 디렉터리는 Cursor AI 에이전트가 프로젝트를 빠르게 이해하도록 돕는 컨텍스트 문서 모음입니다.

## 문서 목록

| 파일 | 설명 |
|------|------|
| [project-overview.md](./project-overview.md) | 프로젝트 목적, 기술 스택, 빌드 환경 |
| [architecture.md](./architecture.md) | 앱 아키텍처, 창 관리, 데이터 흐름 |
| [directory-structure.md](./directory-structure.md) | 디렉터리·파일 구조 |
| [features.md](./features.md) | 기능 모듈별 구현 현황 |
| [conventions.md](./conventions.md) | 코딩 컨벤션 및 패턴 |
| [known-issues.md](./known-issues.md) | 알려진 이슈 및 작업 예정 항목 |

## 사용 방법

- 새 기능 작업 전 `architecture.md`와 `features.md`를 먼저 확인하세요.
- Icon Design 캔버스 관련 작업 시 `known-issues.md`를 참고하세요.
- 파일·모듈 위치는 `directory-structure.md`를 기준으로 합니다.

## 갱신 가이드

프로젝트 구조나 아키텍처가 바뀌면 해당 `.md` 파일을 함께 업데이트하세요.
